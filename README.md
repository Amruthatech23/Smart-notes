##  Features

- Centered smart header  
- Search notes from the top-right corner  
- Enter title and content for new notes  
- Add notes that appear below  
- Edit existing notes  
- Delete notes easily  
- Stylish, responsive, and compact design  
- Optimized for a phone-style interface
